# Product Management System

A comprehensive product management system built with modern web technologies.

## Tech Stack

### Backend
- Node.js
- Express.js
- PostgreSQL (Database)
- Prisma ORM (Database ORM)

### Frontend
- React with Vite
- Tailwind CSS

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn
- PostgreSQL database
